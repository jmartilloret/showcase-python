Basilo: Plant Simulator

Project Description

This project is a simple simulation written in Python that models the growth of a plant named Basilo over 30 days. Each day, random weather conditions (sunny, cloudy, rainy) and wind are generated. These conditions, along with the soil moisture level, affect how many leaves and flowers Basilo has. The goal was to practice using the random module, conditional structures (if/elif/else), loops (while), and basic variable manipulation to create a small dynamic system. Run the code to see how Basilo fares!

Code and Interactive Execution

This is the source code for the Basilo simulation. Press the Run Code button to simulate the 30 days.

import random

daily_weather = ["sunny","partly cloudy","rainy","normal"]
daily_wind = [0,1,2,3,4]
leaves = 0; flowers = 0; day = 1; week = 7; month = 30
soil = 3; MAX_SOIL = 20; MIN_SOIL = 1

def run_simulation():
    for i in range(1, leaves + 1): print(" " * (leaves - i), end=""); print("üåø " * i)
    if flowers > 0: print(" " * (leaves - 1), end=""); print("üíÆ " * flowers)
    print(" " * (leaves - 1), end=""); print("ü´†")
    print("üí© " * soil)

while day <= month:
    for i in range(month):
        if MIN_SOIL > soil: soil = 0
        if MAX_SOIL < soil: soil = 20
        wind = random.choice(daily_wind); weather = random. choice(daily_weather)
        if soil < 3: leaves = leaves - 1; flowers = flowers - 1
        if soil >= 28: flowers = flowers + 2
        if weather == "sunny":
            if wind < 4: print("Day", day," was a nice day, +2 leaves"); leaves += 2; flowers += 1; soil -= 1
            else: leaves += 1; soil -= 3
            print()
        elif weather == "rainy":
            if wind < 3: print("Day",day,"was rainy, Basilo lost 1 leaves"); leaves -=1; soil += 3; print()
            else: print("Thunderstorm on day",day,", bad luck, - 2 leaves."); leaves -= 2; flowers -= 1; soil += 1
        elif weather == "partly cloudy":
            if wind < 3: leaves += 1
            else: leaves -= 1
        else: print("It was a boring day... nothing happened")
        print("Today was",weather,", the wind was",wind,"and the soil is at",soil,".")
        day += 1
        if leaves <= 0: leaves = 1

print(" BASILO WAS BORN! ü´†"); run_simulation(); print()
print(f"Leaves =",abs(leaves), "Flowers =",abs(flowers),"soil =",abs(soil))
